# 飞牛NAS部署Web-Scrcpy指南

## 项目介绍
Web-Scrcpy是一个基于Web的Android设备远程控制工具，允许用户通过浏览器控制和查看Android设备的屏幕，支持屏幕录制、截图、全屏模式等功能。

## 部署步骤

### 方法一：使用Docker Compose（推荐）

1. **准备Docker Compose文件**
   - 将 `docker-compose.yml` 文件复制到飞牛NAS的任意目录中

2. **启动服务**
   - 打开飞牛NAS的Docker套件
   - 进入「容器」页面
   - 点击「添加」按钮
   - 选择「从文件创建」
   - 上传 `docker-compose.yml` 文件
   - 点击「确定」开始部署

### 方法二：手动创建Docker容器

1. **拉取镜像**
   - 打开飞牛NAS的Docker套件
   - 进入「注册表」页面
   - 搜索 `baixin1228/web-scrcpy`
   - 选择最新版本并点击「拉取」

2. **创建容器**
   - 进入「容器」页面
   - 点击「添加」按钮
   - 选择 `baixin1228/web-scrcpy` 镜像
   - 点击「下一步」
   - 配置容器名称为 `web-scrcpy`
   - 点击「下一步」
   - 配置端口映射：主机端口 `8080` 映射到容器端口 `8080`
   - 点击「下一步」
   - 配置环境变量：添加 `TZ=Asia/Shanghai`
   - 点击「下一步」
   - 配置卷：添加 `./data` 映射到 `/app/data`
   - 点击「下一步」
   - 勾选「使用高权限执行容器」
   - 点击「下一步」
   - 点击「完成」创建容器

## 访问和使用

1. **启动服务**
   - 在飞牛NAS的Docker容器列表中，找到 `web-scrcpy` 容器
   - 点击「启动」按钮

2. **访问服务**
   - 打开浏览器
   - 访问 `http://[飞牛NAS的IP地址]:8080`
   - 例如：`http://192.168.1.100:8080`

3. **连接Android设备**
   - 将Android设备通过USB连接到飞牛NAS
   - 确保设备已启用「开发者选项」和「USB调试」
   - 在Web界面中，点击「连接设备」按钮
   - 选择要连接的设备

4. **使用功能**
   - **屏幕镜像**：实时查看设备屏幕
   - **远程控制**：使用鼠标和键盘控制设备
   - **屏幕录制**：点击录制按钮开始录制屏幕
   - **截图**：点击截图按钮捕获当前屏幕
   - **全屏模式**：点击全屏按钮进入全屏模式

## 常见问题

1. **无法连接设备**
   - 确保设备已启用「开发者选项」和「USB调试」
   - 确保USB连接正常
   - 尝试重新插拔USB线缆

2. **屏幕卡顿**
   - 检查网络连接是否稳定
   - 尝试降低视频质量设置

3. **服务无法启动**
   - 检查端口 `8080` 是否被占用
   - 检查Docker容器日志以获取详细错误信息

## 维护建议

1. **定期更新镜像**
   - 定期检查并更新 `baixin1228/web-scrcpy` 镜像到最新版本

2. **备份配置**
   - 定期备份 `./data` 目录中的配置文件

3. **安全注意事项**
   - 建议在局域网内使用，避免暴露到公网
   - 如果需要远程访问，建议使用VPN或其他安全方式
